User interface

The figure below shows how CDH2 is displayed after system startup. The screen shows the CDH2 ‘Dashboard’ with its set of basic functions. These functions are not only available from the menu, but also as tiles in the viewing & processing pane.

  • Click on a tile to change to the list view of the respective function.

The web user interface of CDH2 is basically subdivided into the following areas (each pane or menu is highlighted in color in the figure below):

UserManagement_Overview_01_v1

The side navigation pane

The green highlighted area contains various CDH2 functions for users and administrators. Depending on your role – either an administrator role or a user role like data steward – a different set of functions is displayed here. Click an option in the menu area to change to the associated view in the viewing & processing pane (highlighted in red).

The viewing & processing pane

The area highlighted in red is the viewing & processing pane. Depending on the role you selected in the top menu area (User or Administrator), the associated overview provides you with information either for users or administrators. You can select and edit detail information from these overviews.

The top menu

The top menu (orange) in the header section of the page contains various top menu items. From here, you can open the CDH2 online help and retrieve version or information about your own profile. Other functions in the header allow you to define language settings or log off from CDH2.

How to select items from tables in CDH2

In CDH2 all tables, that provides a selection feature for single rows as well as for multiple rows, are following the same pattern of use. In this example the user management table from the CDH2 Administration page was chosen.

Only 10 table items out of a bigger number of items can be displayed on a single page. On top of the heading row of a table, an information line is displayed. It shows the number of items on the current page as well as the total number of items in the whole table.

You can select all items on the current page by activating the check box in the heading row. The heading row will be displayed on each page of that table.

To jump to the next or the previous page of a table click on the arrow buttons at the bottom of each page.

Item selection procedure

  1. Click on a check box in the first table cell of an item to select a single item from the table.
  2. Click on the check box in the heading row of the table to select all items on the current page.
  3. Click on the link 'Select all <total number> items' to select all items from the whole table. This feature is only available after you clicked the check box in the heading row.
  4. Click on 'Cancel selection' to deselect the previous selection. This feature is only available after you clicked on 'Select all <total number> items' in the previous step.
