User management

With the CDH2 user management you can add and remove users, change their settings, or upload an entire list of users from an external source.

  • The CDH2 user management displays all your users in a single table.
  • To access the CDH2 user management click on 'Administration' in the top menu of the CDH2 page.
  • Click on 'User management' in the 'Administration' menu on the left hand side of the screen.

Important notice:

You will need administrator rights to get access to the CDH2 user management function. Without these rights your access will be denied.

How to manage your users?

The CDH2 user management provides you with the following functions:

Add user:

Click the 'Add user' button to open the 'Add user' page. There you can define all necessary settings for new user that shall be added to your existing users.

Import users:

By clicking 'Import users' you are going to open the 'Import new user' page. You can upload entire lists of users from a MS Excel file into your user management.

Export users:

tbd.

Edit user:

an 'Edit' button is attached to each line. Click on the button to open the 'Edit user' page. There you can change all necessary settings for the respective user.

Delete:

You can delete users by selecting them from the list. Activate the check box in the first table cell of a user. Click on the 'Delete' button to delete all selected users in one single step. To delete a single user you can also click on one of the 'Delete' buttons that are attached to each line. These 'Delete' buttons are characterized with the paper bucket symbol.

Deactivate:

You can deactivate a user's account by selecting them from the list. Activate the check box in the first table cell of a user. Click on the 'Deactivate' button to deactivate all selected user accounts in one single step.

Sort filter for each Column:

The list of users can be sorted by using the sort filter applied to each column of the list. Click on the 'Up' or 'Down' arrow button next to each column heading. The arrow buttons will only be visible, when your mouse cursor hovers over the respective column heading. The list can be sorted in ascending or descending order either alphabetically, with fields containing strings, or by number or date.

Free text search filter:

Enter a search string to filter your list. The list will be filtered according to the entry in the input line. Thus, a smaller portion of the entire list will be displayed as a result.

Pagination settings:

Information about pagination and side navigation are displayed in the bottom line of the page. From any page you can jump to the next or the previous page by clicking the arrow buttons. The total number of pages is displayed as well as the current page. You can jump to a specific page by entering the respective page number in field with current page number. Additionally you can define how many rows of your user table should appear on your page.

How to add a user?

As a system administrator or access manager you will have to add user account to your CDH2 user management. In this case, you have to assign certain settings to a specific user account. A user can see these settings in his own user profile. Within narrow restriction a user will be able to overwrite your settings by his own settings, like defining a new password.

Use the 'Add user' function to add a new user account to your user management.

  • Click on the 'Add user' button in the user management view of the CDH2 administration. The 'Add user' page will be displayed.
  • Enter the necessary user credentials into the input fields of the 'Add user' page.
  • Select the respective localization entries for the user from the drop-down fields.
  • Select at least one role that apply to the respective user from the drop-down field.
  • Select the team that belongs to the respective user from the drop-down field.
  • Define whether the new account is supposed to be active, inactive or intended for being valid for limited period of time.
  • Click on 'Add user' to add the user to your user management table and to submit a notification to this user.
  • Click 'Cancel' to abort and get back to the user management view.

As a result a user record will be created and added to the list view in your user management view. The user will be notified automatically by an e-mail. This e-mail prompts the user to define a new password for his or her account.

Click on the required radio button to define, whether the user account has to be active, inactive or will expire after a specific period of time. If the account is supposed to expire enter an expiration date in the field ‘Expiration date’.
How to edit the settings of an existing user?

As a system administrator or access manager you might have to change the settings for an individual user account. After these changes a user will see these settings in his own user profile.

Use the 'Add user' function to add a new user account to your user management.

  • Click on the 'Edit' button that is attached to each user record in the user management view. The 'Edit user' page will be displayed.
  • Change the required user credentials in the input fields of the 'Edit user' page.
  • Choose different localization entries for the respective user in the drop-down fields.
  • Choose a different role or add a new role to the user's profile from the drop-down field.
  • Choose a different team or add a new team to the user's profile from the drop-down field.
  • Define whether the account is supposed to be active, inactive or intended for being valid for limited period of time.
  • Click on 'Save' to confirm the changed settings.
  • Click 'Revert changes' to restore the previous settings.

As a result the changed user record will appear in the list view of your user management. The user will not be notified about this changes. However, the user will be able to see the new settings in his user profile.

How to reset a user's password?

The reset of a user's account password is triggered by a request for password reset. A user will send this request in case he has forgotten his password. As a system administrator or access manager you will be notified by an e-mail about this incident. The user's password needs to be reset, so that the user is able to define a new password.

These are your next steps:

  • Choose 'Administration' in the top menu of CDH2.
  • Click on 'User Management' in the administration menu.
  • Select the user, who has forgotten his password from the user list. You can use the search line to filter the list by entering the user's name.
  • Click on the 'Edit' button attached to the user. The 'Edit users' page will be opened.
  • Click on the 'Reset password' button in the 'Reset password' section of the page. The user will get an e-mail notification and a request to define a new account password.

How to delete a user?

Three different ways allows you to delete a user and his/her account from your user management.

  • Choose 'Administration' in the top menu of CDH2.
  • Click on 'User Management' in the administration menu.
  • Select the all users, you want to delete from the user list. Click on the selection check box in the first columns of the table. You can also use the search line to filter the list.

Now there are three options:

  1. Click on the 'Delete' button on top of the user list view. All selected previously selected users will be deleted.
  2. Click on the 'Delete' button attached to the user line. Only this single user will be deleted
  3. Click on the 'Edit' button attached to the user. The 'Edit users' page will be opened. Click on the 'Delete user' button in the 'Delete user' section of this page. Only this single user will be deleted.

The user will be removed from the user list.
