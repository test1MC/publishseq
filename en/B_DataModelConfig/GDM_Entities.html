Entities

Within the Customer Data Hub self defined entities are provided. Entities are a collection of values that uniquely define a data object, like postal addresses, phone numbers, or bank accounts. Entities contain several data fields. Not all of them are always present in an actual entity, but as a whole these fields uniquely identify a single address, a phone number or a bank account.

The self defined data fields of an entity are supposed to match the corresponding definitions of the data model in your source system.

The following entities are available within CDH2:

  • PERSON
  • ORGANIZATION
  • LOCATION
  • PHONE
  • EMAIL
  • WEBSITE
  • META

Expand the drop-down fields below this section for detailed information about the data fields of an entity.
